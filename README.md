## Installation :

Créez un fichier nommé /root/start.sh et ajoutez y:

```sh
#!/bin/bash

cd /root/python/

python main.py
```

Ensuite, créez le répertoire /root/python/  et /root/python/update/ pour les mises à jours en exécutant la commande:

```sh
$ mkdir /root/python/ /root/python/update/
```
